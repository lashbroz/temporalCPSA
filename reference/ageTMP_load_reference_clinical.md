# Load reference-cohort clinical data

Public-facing alias for
[`ageTMP_load_cdisc_clinical()`](https://lashbroz.github.io/temporalCPSA/reference/ageTMP_load_cdisc_clinical.md).
In the manuscript source files, the external/reference survival cohort
is labeled `cDisc`; in the package API, "reference cohort" is the
preferred general term.

## Usage

``` r
ageTMP_load_reference_clinical(data_dir = "data")
```

## Arguments

- data_dir:

  Path to the public data directory.

## Value

A clinical data frame.
