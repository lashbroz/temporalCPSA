# Load membrane glycopeptide annotation

This documented package data source replaces the manuscript
`membrane_glycopeptide.RData` workspace for glycopeptide-level survival
table reproduction. It contains glycopeptides annotated as
membrane-localized in the original manuscript workflow.

## Usage

``` r
ageTMP_load_glyco_membrane_features()
```

## Value

A data frame with `glycopeptide` and `membrane.loc` columns.
