# Load discovery-cohort clinical data

Public-facing alias for
[`ageTMP_load_cdisc_clinical()`](https://lashbroz.github.io/temporalCPSA/reference/ageTMP_load_cdisc_clinical.md).

## Usage

``` r
ageTMP_load_discovery_clinical(data_dir = "data")
```

## Arguments

- data_dir:

  Path to the public data directory.

## Value

A clinical data frame.
